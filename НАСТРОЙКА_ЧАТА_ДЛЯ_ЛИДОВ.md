# Как настроить отдельный чат для уведомлений о лидах

## Сейчас уведомления приходят вам в личку

Если вы хотите чтобы уведомления о лидах приходили в **отдельный канал/группу** (а не в личные сообщения), выполните следующие шаги:

## Пошаговая инструкция

### Шаг 1: Создайте канал или группу

1. Откройте Telegram
2. Создайте **приватный канал** или **группу**
3. Назовите его например "Лиды Legal AI Bot"

### Шаг 2: Добавьте бота в канал/группу

1. Откройте созданный канал/группу
2. Нажмите на название → **Управление**
3. Выберите **Администраторы** → **Добавить администратора**
4. Найдите вашего бота (имя бота из @BotFather)
5. Дайте ему права **"Публиковать сообщения"**

### Шаг 3: Получите ID канала/группы

**Способ 1 (самый простой) - через @userinfobot:**

1. Добавьте бота **@userinfobot** в ваш канал/группу
2. Он автоматически отправит сообщение с ID
3. Скопируйте число (например: **-1001234567890**)
4. Можете удалить @userinfobot из канала

**Способ 2 - через API:**

1. Отправьте любое сообщение в ваш канал/группу
2. Откройте в браузере:
   ```
   https://api.telegram.org/bot<ВАШ_BOT_TOKEN>/getUpdates
   ```
3. Найдите в JSON: `"chat":{"id": -1001234567890}`
4. Скопируйте это число

**ВАЖНО:** ID канала/группы обычно начинается с **-100**

### Шаг 4: Добавьте в .env

1. Откройте файл `.env` в редакторе:
   ```bash
   nano .env
   ```

2. Добавьте строку (или раскомментируйте если есть):
   ```env
   LEADS_CHAT_ID=-1001234567890
   ```

   Замените `-1001234567890` на ваш реальный ID!

3. Сохраните файл (Ctrl+O, Enter, Ctrl+X)

### Шаг 5: Перезапустите бота

```bash
# Остановите бота (Ctrl+C)
# Или если запущен через systemd:
sudo systemctl restart telegram-bot

# Или запустите вручную:
./start.sh
```

## Проверка

1. Попросите кого-то написать боту (или создайте тестовый аккаунт)
2. Пообщайтесь с ботом, укажите какие-то данные
3. Уведомление должно прийти в **ваш канал/группу**, а **НЕ в личку** ✅

## Что изменилось в последнем обновлении

✅ **Исправлено:** Уведомления отправляются **ОДИН РАЗ** (а не 5 раз как было)
✅ **Добавлено:** Telegram username и ID в уведомлении
✅ **Сохранено:** Ваши сообщения НЕ создают лиды (только реальные пользователи)

## Если что-то не работает

**Уведомления не приходят в канал:**
- Проверьте что бот добавлен как администратор
- Проверьте что у бота есть права "Публиковать сообщения"
- Проверьте правильность ID (должен начинаться с `-100`)
- Перезапустите бота после изменения .env

**Уведомления всё еще приходят в личку:**
- Убедитесь что LEADS_CHAT_ID прописан в .env
- Убедитесь что бот перезапущен после изменения .env
- Проверьте логи: `tail -f logs/bot.log`

**Получаю много уведомлений об одном лиде:**
- Это исправлено в последнем обновлении
- Обновите код: `git pull origin main`
- Перезапустите бота
